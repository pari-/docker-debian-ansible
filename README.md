## General

The image is based on Debian GNU/Linux (release: stretch) and contains Ansible (located in /opt/) in the following version flavour:
  - "2.7.9.0"

The automated build(s) can be found here: [irap/docker-debian-ansible](https://hub.docker.com/r/irap/docker-debian-ansible/)
